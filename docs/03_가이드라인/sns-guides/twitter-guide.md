## 2. 핵심 전략

### 외부 링크 = 노출 억제

> ⚠️ **본문에 외부 링크를 넣으면 노출이 억제됩니다!**

| ❌ 잘못된 방법 | ✅ 올바른 방법 |
|-------------|-------------|
| 본문에 URL 삽입 | **리플라이에 URL 배치** |
| "링크 클릭하세요" | "프로필 링크 확인" |

### 트윗스톰 (스레드)

280자 제한을 넘어 상세 콘텐츠 전달:

```
[TWEET 1 — 후킹] ≤280자 + "🧵 스레드 ↓" + 대표 이미지 1장
[TWEET 2~3 — 본문] ≤280자 + 이미지 2~4장
[TWEET 4 — CTA] ≤280자 + 해시태그 (글자 수 포함!)
```

### SimClusters 최적화

- X는 유저와 콘텐츠를 **관심사 클러스터**로 분류
- 니치(전문 분야) 일관성 유지 → 정확한 관객 매칭
- 갑작스러운 주제 전환 → 클러스터 혼란 → 도달 하락

## 4. 체크리스트

### ✅ 해야 할 것
- 첫 문장 후킹 → 드웰타임 확보
- 네이티브 미디어(직접 업로드) 사용
- 링크는 리플라이에 배치
- 니치 일관성 유지
- 게시 후 즉시 대화 참여
- 하루 3~5개 고품질 포스트 (2~3시간 간격)

### ❌ 하지 말아야 할 것
- 본문에 외부 링크 삽입 (노출 억제)
- 주제 갑작스런 전환
- 자동화 응답 / "팔로우 감사" 류
- 품질 없는 대량 포스팅

> **핵심:** 2026년 X는 **"pay-to-play"** 환경. X Premium + 롱폼 고품질 콘텐츠 + 대화 유도가 필수.
