## 2. 핵심 전략

### 릴스(Reels) 최우선

- 숏폼 비디오(릴스)가 **유기적 도달 최대 채널**
- **60초 이내** 권장, 세로 **9:16** (1080×1920)
- 첫 3초 후킹 필수 → 완주율이 핵심
- 타 플랫폼 워터마크 콘텐츠 = 다운랭크

### 80/20 법칙

| 비율 | 콘텐츠 유형 |
|------|------------|
| **80%** | 가치 제공 (교육, 엔터테인먼트, 영감) |
| **20%** | 프로모션/세일즈 |

### 커뮤니티 그룹 활용

- 비즈니스 페이지에 **Facebook 그룹 연동** → 그룹 포스트가 페이지보다 도달 유리
- 그룹 내 대화 → 알고리즘 호감도 ↑

### 모바일 최적화 (텍스트 잘림)

```
[0~80자]    ← 모바일 피드에서 바로 보이는 영역 (후킹!)
[80~125자]  ← 일부 디바이스에서 보일 수 있음
[125자~]    ← "더 보기" 클릭 필요 → 대부분 안 읽음
```

→ **첫 40~80자에 핵심 메시지** 배치 필수

## 3. 기술 사양

| 항목 | 값 |
|------|-----|
| 글자 수 | 최대 **63,206자** (권장 40~80자 후킹) |
| 이미지 (캐러셀) | **최대 10장** |
| 이미지 파일 크기 | 30 MB |
| 이미지 비율 | **4:5** (1080×1350) 또는 **1:1** (1080×1080) |
| 동영상 최대 길이 | **240분** (릴스 60초 권장) |
| 동영상 파일 크기 | 10 GB (권장 4 GB) |
| 동영상 형식 | MP4, MOV |
| 릴스 비율 | **9:16** (1080×1920) |

## 4. 체크리스트

### ✅ 해야 할 것
- 릴스(세로 9:16) 60초 이내 → 최대 유기적 도달
- 첫 40~80자에 핵심 후킹
- 질문/투표 → 의미있는 댓글 유도
- 커뮤니티 그룹 연동
- 주 3~5회 고품질 포스팅
- 댓글에 수 시간 내 응답
- 직원/파트너 공유 유도 (개인 계정 공유 = 도달 ↑)

### ❌ 하지 말아야 할 것
- 타 플랫폼 워터마크 리포스트
- 어그로성 참여 유도 ("좋아요 누르면 ~")
- 주제 산발적 전환
- 매일 저품질 대량 포스팅

> **핵심:** Facebook은 **"공유"와 "대화"** 가 도달을 결정하는 플랫폼. 릴스 + 커뮤니티 그룹이 2026년 유기적 성장의 양대 축.
