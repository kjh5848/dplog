# 카카오 로그인 API 명세서

## 엔드포인트 개요

### Base URL
```
http://localhost:8081  (개발환경)
```

### 인증 방식
- 세션 기반 인증 (JSESSIONID 쿠키)
- CORS 허용 도메인: `http://localhost:3000`

## API 명세

### 1. 카카오 로그인 콜백

#### 기본 정보
- **URL**: `/v1/auth/kakao/callback`
- **Method**: `POST`
- **Content-Type**: `application/json`
- **인증**: 불필요 (로그인 API)

#### 요청 (Request)

**Headers:**
```http
Content-Type: application/json
Cookie: JSESSIONID={세션ID}  # 새 세션 생성을 위해 포함
```

**Body:**
```json
{
  "code": "카카오에서 전달받은 인증 코드",
  "state": "CSRF 방지를 위한 상태값"
}
```

**Request DTO (ReqKakaoCallbackDTOApiV1):**
```java
public class ReqKakaoCallbackDTOApiV1 {
    @NotBlank(message = "카카오 인증 코드는 필수입니다.")
    private String code;
    
    @NotBlank(message = "State 파라미터는 필수입니다.")
    private String state;
}
```

#### 응답 (Response)

**성공 응답 (200 OK):**
```json
{
  "code": 0,
  "message": "카카오 로그인에 성공했습니다.",
  "data": {
    "user": {
      "id": 123,
      "username": "kakao_123456789",
      "name": "홍길동",
      "email": "hong@example.com",
      "profileImage": "https://k.kakaocdn.net/dn/profile.jpg",
      "authority": ["USER"],
      "provider": "KAKAO"
    },
    "isNewUser": true
  }
}
```

**Response DTO (ResKakaoLoginDTOApiV1):**
```java
public class ResKakaoLoginDTOApiV1 {
    private UserInfo user;
    private Boolean isNewUser;
    
    public static class UserInfo {
        private Long id;
        private String username;
        private String name;
        private String email;
        private String profileImage;
        private List<UserAuthoritySort> authority;
        private String provider;
    }
}
```

#### 에러 응답

**1. 중복 로그인 (400 Bad Request):**
```json
{
  "code": -1,
  "message": "이미 로그인 중인 계정입니다.",
  "data": null
}
```

**2. 만료된 계정 (400 Bad Request):**
```json
{
  "code": -1,
  "message": "만료된 계정입니다.",
  "data": null
}
```

**3. 사용중지된 계정 (400 Bad Request):**
```json
{
  "code": -1,
  "message": "사용중지된 계정입니다.",
  "data": null
}
```

**4. 카카오 API 키 설정 오류 (401 Unauthorized):**
```json
{
  "code": -9,
  "message": "카카오 API 키 설정을 확인해주세요.",
  "data": null
}
```

**5. 시스템 오류 (500 Internal Server Error):**
```json
{
  "code": -99,
  "message": "카카오 로그인 처리 중 시스템 오류가 발생했습니다.",
  "data": null
}
```

**6. 입력값 검증 오류 (400 Bad Request):**
```json
{
  "code": -3,
  "message": "카카오 인증 코드는 필수입니다.",
  "data": null
}
```

#### 세션 처리

**성공 시 세션 생성:**
- `JSESSIONID` 쿠키 자동 설정
- 세션 속성에 `authInfo` 저장
- SessionManager에 사용자 세션 등록

**세션 쿠키 속성:**
```http
Set-Cookie: JSESSIONID=ABC123DEF456; 
            Max-Age=2600000; 
            Path=/; 
            HttpOnly; 
            SameSite=Lax
```

## 프론트엔드 연동 가이드

### 1. 카카오 로그인 시작

```javascript
// 1. 카카오 OAuth URL 생성
const kakaoAuthUrl = `https://kauth.kakao.com/oauth/authorize?` +
  `client_id=${KAKAO_REST_API_KEY}&` +
  `redirect_uri=${encodeURIComponent('http://localhost:3000/kakao/callback')}&` +
  `response_type=code&` +
  `state=${generateRandomState()}`;

// 2. 카카오 로그인 페이지로 리다이렉트
window.location.href = kakaoAuthUrl;
```

### 2. 콜백 처리

```javascript
// /kakao/callback 페이지에서
const urlParams = new URLSearchParams(window.location.search);
const code = urlParams.get('code');
const state = urlParams.get('state');

if (code) {
  // 백엔드 API 호출
  const response = await fetch('/v1/auth/kakao/callback', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
    },
    credentials: 'include', // 쿠키 포함
    body: JSON.stringify({ code, state })
  });

  const result = await response.json();
  
  if (result.code === 0) {
    // 로그인 성공
    console.log('사용자 정보:', result.data.user);
    console.log('신규 사용자 여부:', result.data.isNewUser);
    
    // 메인 페이지로 리다이렉트
    window.location.href = '/dashboard';
  } else {
    // 로그인 실패
    alert(result.message);
  }
}
```

### 3. 로그인 상태 확인

```javascript
// 페이지 로드 시 로그인 상태 확인
const checkAuthStatus = async () => {
  try {
    const response = await fetch('/v1/auth/info', {
      credentials: 'include'
    });
    
    const result = await response.json();
    
    if (result.code === 0) {
      // 로그인됨
      return result.data.user;
    } else {
      // 로그인되지 않음
      return null;
    }
  } catch (error) {
    console.error('인증 상태 확인 실패:', error);
    return null;
  }
};
```

### 4. 로그아웃

```javascript
const logout = async () => {
  try {
    const response = await fetch('/v1/auth/logout', {
      method: 'POST',
      credentials: 'include'
    });
    
    const result = await response.json();
    
    if (result.code === 0) {
      // 로그아웃 성공
      window.location.href = '/login';
    }
  } catch (error) {
    console.error('로그아웃 실패:', error);
  }
};
```

## 환경 설정

### 백엔드 설정 (application-dev.yml)

```yaml
kakao:
  rest-api-key: ${KAKAO_REST_API_KEY:your_kakao_rest_api_key}
  client-secret: ${KAKAO_CLIENT_SECRET:}
  redirect-uri: ${KAKAO_REDIRECT_URI:http://localhost:3000/kakao/callback}

server:
  port: 8081
  servlet:
    session:
      timeout: 2600000  # 43분
      cookie:
        max-age: 2600000
        same-site: lax
        secure: false  # 개발환경
```

### 프론트엔드 환경변수

```javascript
// .env 또는 config.js
const config = {
  KAKAO_REST_API_KEY: 'your_kakao_rest_api_key',
  BACKEND_URL: 'http://localhost:8081',
  REDIRECT_URI: 'http://localhost:3000/kakao/callback'
};
```

## 테스트 방법

### cURL을 이용한 API 테스트

```bash
# 1. 카카오 콜백 API 테스트
curl -X POST http://localhost:8081/v1/auth/kakao/callback \
  -H "Content-Type: application/json" \
  -d '{
    "code": "test_authorization_code",
    "state": "test_state"
  }' \
  -v

# 2. 로그인 상태 확인 (세션 쿠키 포함)
curl -X GET http://localhost:8081/v1/auth/info \
  -H "Cookie: JSESSIONID=your_session_id" \
  -v
```
