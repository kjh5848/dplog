# 연관검색어 기능 명세서

## 1. 기능 개요

### 1.1 목적
네이버 플레이스 순위 서비스에서 사용자가 입력한 키워드와 연관된 검색어를 제공하여, 보다 효과적인 SEO 전략 수립을 지원한다.

### 1.2 기능 설명
- 사용자가 입력한 키워드를 기반으로 네이버 검색광고 API를 활용하여 연관 검색어를 추천
- 키워드별 검색량, 경쟁강도 등 부가 정보 제공
- 멤버십 등급에 따른 사용 횟수 제한 관리
- 검색 이력 및 사용량 추적

## 2. 기능 요구사항

### 2.1 기본 기능
#### 2.1.1 연관검색어 검색
- **기능**: 입력된 키워드의 연관검색어 목록 조회
- **입력**: 
  - 키워드 (1-5개, 각 키워드 최대 길이 제한)
  - 요청 타입 (연관검색어/키워드분석)
- **출력**:
  - 연관검색어 목록
  - 검색량 정보
  - 경쟁강도
  - 클릭률 정보

#### 2.1.2 키워드 분석
- **기능**: 개별 키워드에 대한 상세 분석 정보
- **입력**: 키워드 문자열 (개행문자로 구분된 다중 키워드)
- **출력**: 키워드별 상세 분석 데이터

#### 2.1.3 블로그 검색 정보
- **기능**: 특정 키워드의 블로그 검색 결과 정보
- **입력**: 단일 키워드
- **출력**: 블로그 검색 관련 통계 정보

### 2.2 비기능적 요구사항

#### 2.2.1 성능 요구사항
- API 응답시간: 평균 3초 이내
- 동시 사용자 처리: 최대 100명
- 네이버 API 호출 최적화를 통한 응답시간 단축

#### 2.2.2 보안 요구사항
- 사용자 인증 필수 (세션 기반)
- API 호출 로깅 및 감사
- 민감한 API 키 보안 관리

#### 2.2.3 가용성 요구사항
- 서비스 가용성: 99.9%
- 장애 발생 시 자동 복구 메커니즘
- 외부 API 장애 시 적절한 오류 처리

## 3. API 명세서

### 3.1 연관검색어 조회 API

#### 3.1.1 기본 정보
- **URL**: `GET /v1/nplace/keyword/nsearchad/keywordstool/relation`
- **인증**: 세션 기반 인증 필요
- **사용량 제한**: 멤버십별 월간 사용 횟수 제한

#### 3.1.2 요청 파라미터
```json
{
  "keywordList": ["string"], // 1-5개 키워드 배열
  "requestType": "RELATION",  // RELATION 또는 KEYWORD
}
```

#### 3.1.3 응답 데이터
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "keywordList": [
      {
        "relKeyword": "string",      // 연관 키워드
        "monthlyPcQcCnt": number,    // 월간 PC 검색수
        "monthlyMobileQcCnt": number, // 월간 모바일 검색수  
        "monthlyAvePcClkCnt": number, // 월간 평균 PC 클릭수
        "monthlyAveMobileClkCnt": number, // 월간 평균 모바일 클릭수
        "monthlyAvePcCtr": number,    // 월간 평균 PC CTR
        "monthlyAveMobileCtr": number, // 월간 평균 모바일 CTR
        "plAvgDepth": number,         // 평균 노출 순위
        "compIdx": "string"           // 경쟁 강도
      }
    ]
  }
}
```

### 3.2 키워드 분석 API

#### 3.2.1 기본 정보
- **URL**: `POST /v1/nplace/keyword/nsearchad/keywordstool`
- **인증**: 세션 기반 인증 필요
- **사용량 제한**: 멤버십별 월간 사용 횟수 제한

#### 3.2.2 요청 데이터
```json
{
  "nplaceKeywordNsearchadKeywordstoolKeyword": {
    "keywordString": "string" // 개행문자로 구분된 키워드 목록
  }
}
```

#### 3.2.3 응답 데이터
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "keywordList": [
      // 키워드별 분석 결과 (연관검색어 조회와 동일한 구조)
    ]
  }
}
```

### 3.3 블로그 검색 정보 API

#### 3.3.1 기본 정보
- **URL**: `GET /v1/nplace/keyword/nblog/search/info`
- **인증**: 불필요
- **사용량 제한**: 없음

#### 3.3.2 요청 파라미터
```json
{
  "keyword": "string" // 검색할 키워드
}
```

#### 3.3.3 응답 데이터
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "blogSearchResult": {
      // 블로그 검색 결과 정보
    }
  }
}
```

    - CTR 정보

  - 결과 내보내기 기능 (Excel/CSV)

#### 5.1.3 검색 이력 화면

  - 최근 검색 키워드 목록
  - 검색 날짜/시간

  - 이력 삭제 기능

### 5.2 사용자 인터랙션

#### 5.2.1 키워드 입력
- 자동완성 기능

- **FREE**: 월 5회 제한
- **BASIC**: 월 50회 제한  
- **PREMIUM**: 월 200회 제한
- **ENTERPRISE**: 무제한

### 6.2 사용량 계산
- 월별 사용량 초기화 (매월 1일 0시)
- API 호출 성공 시에만 사용량 차감
