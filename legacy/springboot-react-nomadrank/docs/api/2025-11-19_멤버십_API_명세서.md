## 2. 현재 구독 정보 조회
**GET** `/v1/membership/current?userId={userId}`

### 요청 파라미터
- `userId` (optional): 조회할 사용자 ID (없으면 본인 조회)

### 응답
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "currentSubscription": {
      "id": 15,
      "userId": 123,
      "membership": {
        "id": 2,
        "name": "PRO",
        "point": 0,
        "price": 29000,
        "description": "개인 사업자를 위한 멤버십입니다.",
        "isPopular": true,
        "colorScheme": "blue",
        "sortOrder": 1
      },
      "startDate": "2025-01-01",
      "endDate": "2025-02-01",
      "membershipState": "ACTIVATE",
      "remainingDays": 22
    }
  }
}
```

## 5. 가격대별 멤버십 조회
**GET** `/v1/membership/price-range?minPrice={minPrice}&maxPrice={maxPrice}`

### 요청 파라미터
- `minPrice`: 최소 가격
- `maxPrice`: 최대 가격

### 응답
```json

  "message": "success",
  "data": {
    "membershipList": [
      {
        "id": 2,
        "name": "PRO",
        "point": 0,
        "price": 29000,
        "description": "개인 사업자를 위한 멤버십입니다.",
        "isPopular": true,
        "colorScheme": "blue",
        "sortOrder": 1
      }
    ]

}
```

## 공통 오류 응답
### 인증 오류 (401)
```json
{
  "code": 401,
  "message": "인증이 필요합니다.",
  "data": null
}
```

### 멤버십을 찾을 수 없음 (400)
```json
{
  "code": 400,
  "message": "해당 멤버십이 존재하지 않습니다.",
  "data": null
}
```

### 권한 없음 (403)
```json
{
  "code": 403,
  "message": "접근 권한이 없습니다.",
  "data": null
}
```

## 필드 설명

### Membership 필드
- `id`: 멤버십 고유 ID
- `name`: 멤버십 이름 (FREE, PRO, PREMIUM, AGENCY)
- `point`: 멤버십 포인트 (기존 필드, 호환성 유지)
- `price`: 월 구독 가격 (원) ⭐ 새로 추가
- `description`: 멤버십 상세 설명 ⭐ 새로 추가
- `isPopular`: 인기 멤버십 여부 (UI 배지용) ⭐ 새로 추가
- `colorScheme`: UI 색상 테마 ⭐ 새로 추가
- `sortOrder`: 표시 순서 ⭐ 새로 추가

### MembershipUser 필드
- `startDate`: 멤버십 시작일
- `endDate`: 멤버십 종료일
- `membershipState`: 멤버십 상태 (READY, ACTIVATE, EXPIRED)
- `remainingDays`: 남은 일수

## 참고사항
- 모든 API는 세션 기반 인증이 필요합니다
- 멤버십은 `sortOrder` 순서로 정렬되어 반환됩니다
- `isPopular`가 `true`인 멤버십은 UI에서 "인기" 배지를 표시할 수 있습니다
- 기존 `point` 필드는 호환성을 위해 유지됩니다
- Plan 시스템이 Membership으로 완전히 통합되었습니다
