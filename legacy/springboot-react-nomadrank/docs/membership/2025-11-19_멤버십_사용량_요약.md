## 1. 테이블 개요

| 컬럼 | 타입 | 널 | 설명 |
| --- | --- | --- | --- |
| id | BIGINT | N | 사용량 고유 ID |
| membership_user_id | BIGINT | N | MEMBERSHIP_USER FK |
| service_sort | VARCHAR(50) | N | 서비스 구분 (ServiceSort) |
| period_type | VARCHAR(20) | N | 집계 단위 (MONTHLY/DAILY) |
| period_key | CHAR(10) | N | 집계 키 (MONTHLY: yyyy-MM, DAILY: yyyy-MM-dd) |
| limit_count | INT | Y | 한도 (NULL=무제한) |
| used_count | INT | N | 현재 사용량 (기본 0) |
| unlimited | TINYINT(1) | N | 무제한 여부 |
| last_used_at | DATETIME | Y | 마지막 사용 시각 |
| created_at | DATETIME | N | 생성일시 |
| updated_at | DATETIME | N | 수정일시 |

- **유니크 키:** `(membership_user_id, service_sort, period_type, period_key)`  
- **인덱스:** `(membership_user_id, service_sort)`  
- **규칙:** 무제한 플랜은 `unlimited=1`, `limit_count=NULL`  
- **기간 규칙:** MONTHLY → `yyyy-MM`, DAILY → `yyyy-MM-dd`

## 2. MySQL DDL

```sql
CREATE TABLE `MEMBERSHIP_USAGE` (
  `id` BIGINT NOT NULL AUTO_INCREMENT COMMENT '사용량 고유 ID',
  `membership_user_id` BIGINT NOT NULL COMMENT '멤버십 가입자 ID (MEMBERSHIP_USER FK)',
  `service_sort` VARCHAR(50) NOT NULL COMMENT '서비스 구분 (ServiceSort)',
  `period_type` VARCHAR(20) NOT NULL COMMENT '집계 단위 (MONTHLY/DAILY 등)',
  `period_key` CHAR(10) NOT NULL COMMENT '집계 기간 키 (예: 2025-10, 2025-10-21)',
  `limit_count` INT NULL COMMENT '서비스 사용 한도 (null: 무제한)',
  `used_count` INT NOT NULL DEFAULT 0 COMMENT '현재 사용량',
  `unlimited` TINYINT(1) NOT NULL DEFAULT 0 COMMENT '무제한 여부 (1: 무제한)',
  `last_used_at` DATETIME NULL COMMENT '마지막 사용 시각',
  `created_at` DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '생성일시',
  `updated_at` DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '수정일시',
  PRIMARY KEY (`id`),
  UNIQUE KEY `uk_membership_usage_period` (`membership_user_id`, `service_sort`, `period_type`, `period_key`),
  KEY `ix_membership_usage_user_sort` (`membership_user_id`, `service_sort`),
  CONSTRAINT `fk_usage_membership_user`
    FOREIGN KEY (`membership_user_id`) REFERENCES `MEMBERSHIP_USER`(`id`)
    ON DELETE CASCADE ON UPDATE RESTRICT
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

## 3. JPA 엔티티

```java
@Entity
@Table(
    name = "MEMBERSHIP_USAGE",
    uniqueConstraints = @UniqueConstraint(
        name = "uk_membership_usage_period",
        columnNames = ('membership_user_id', 'service_sort', 'period_type', 'period_key')
    )
)
@Getter @Setter
@NoArgsConstructor @AllArgsConstructor @Builder
public class MembershipUsageEntity {
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    @ManyToOne(fetch = FetchType.LAZY, optional = false)
    @JoinColumn(name = "membership_user_id", nullable = false,
            foreignKey = @ForeignKey(name = "fk_usage_membership_user"))
    private MembershipUserEntity membershipUser;

    @Enumerated(EnumType.STRING)
    @Column(name = "service_sort", nullable = false, length = 50)
    private ServiceSort serviceSort;

    @Enumerated(EnumType.STRING)
    @Column(name = "period_type", nullable = false, length = 20)
    private UsagePeriodType periodType;

    @Column(name = "period_key", nullable = false, length = 10)
    private String periodKey;

    @Column(name = "limit_count")
    private Integer limitCount;

    @Builder.Default
    @Column(name = "used_count", nullable = false)
    private Integer usedCount = 0;

    @Builder.Default
    @Column(name = "unlimited", nullable = false)
    private Boolean unlimited = false;

    @Column(name = "last_used_at")
    private LocalDateTime lastUsedAt;

    @CreationTimestamp
    @Column(name = "created_at", nullable = false, updatable = false)
    private LocalDateTime createdAt;

    @UpdateTimestamp
    @Column(name = "updated_at", nullable = false)
    private LocalDateTime updatedAt;
}
```

## 4. 서비스 로직 개요

1. **첫 가입 시**
   - 무료 요금제 자동 생성
   - 사용량 레코드 자동 생성 (`limit_count`, `unlimited` 적용)

2. **업그레이드 시**
   - 기존 ACTIVE → EXPIRED
   - 새 플랜 ACTIVE
   - 사용량 초기화 (`used_count=0`)
   - 한도 및 무제한 여부 갱신

3. **다운그레이드 시**
   - 다음달부터 적용 (스케줄러 등록)
   - 다음달 1일에 자동 전환 및 초기화

4. **모든 과정은 트랜잭션 내에서 처리**
   - 외부 결제 성공 웹훅 이후 트랜잭션 확정
