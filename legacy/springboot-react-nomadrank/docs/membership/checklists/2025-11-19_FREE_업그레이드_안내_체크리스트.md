# FREE 업그레이드 안내 문구 체크리스트 (2025-11-19)

1. **정보 수집**
   - [ ] FREE 사용자의 잔여 호출 수/제한 메시지를 현재 어디에서 노출하는지 파악.
   - [ ] 업그레이드 CTA 전·후 어느 위치에 “업그레이드 시 즉시 초기화” 문구를 띄울지 결정.

2. **명세 작성**
   - [ ] 정확한 문구를 확정 (예: “무료 플랜에서 상위 플랜으로 업그레이드하면 사용량이 즉시 초기화됩니다”).
   - [ ] 다국어/표현 방식(문장, 배지, 토스트 등)을 선택.

3. **프런트 구현**
   - [ ] 업그레이드 모달/페이지에 안내 문구 삽입.
   - [ ] FREE 사용자만 조건부로 노출하고 CTA 클릭 전에 동의/확인 UI 추가.

4. **백엔드 표시값**
   - [ ] 필요 시 `SubscriptionUpgradeResponse` 또는 별도 API에 “FREE → 상위 플랜” 여부 플래그 제공.
   - [ ] 안내 문구 표시를 위한 메시지를 API 응답에 포함할지 결정.

5. **QA/검증**
   - [ ] FREE → 상위 플랜 업그레이드 경로 테스트(모달/토스트 노출 여부 확인).
   - [ ] 상위 플랜(실속/성장 등)에서는 문구가 노출되지 않는지 확인.

6. **문서화**
   - [ ] 프런트 요구 사항 문서(예: 20251117 프런트 조치) 업데이트.
   - [ ] CS/FAQ에도 “업그레이드 시 사용량 초기화” 안내 문구 추가.
