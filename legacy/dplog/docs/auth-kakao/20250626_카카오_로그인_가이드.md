# 카카오 로그인 구현 가이드

## 📋 개요

카카오 로그인을 기존 일반 로그인 시스템에 추가하여 사용자가 두 가지 방식으로 로그인할 수 있도록 구현합니다.

---

## 🎯 구현 목표

1. **기존 로그인 유지**: 현재 일반 로그인 시스템은 그대로 유지
2. **카카오 로그인 추가**: REST API를 통한 카카오 로그인 추가
3. **계정 연동**: 기존 사용자도 카카오 계정 연동 가능
4. **신규 사용자**: 카카오 로그인으로 신규 회원가입 가능

---

## 🔑 카카오 개발자 설정

### 1. 카카오 디벨로퍼스 앱 설정

**필수 설정 항목:**

1. **플랫폼 등록**
   ```
   Web 플랫폼: https://dplog.com
   ```

2. **카카오 로그인 활성화**
   - 상품 설정 > 카카오 로그인 > 활성화 설정 > ON

3. **Redirect URI 설정**
   ```
   https://dplog.com/auth/kakao/callback
   ```

4. **동의항목 설정**
   - 닉네임: 필수 동의
   - 프로필 사진: 선택 동의
   - 이메일: 필수 동의 (기존 계정 연동을 위해)

5. **앱 키 정보**
   ```
   REST API 키: 클라이언트에서 사용
   Admin 키: 백엔드 서버에서 사용 (보안 주의)
   ```

### 2. 환경변수 설정

**.env.local (프론트엔드)**
```env
NEXT_PUBLIC_KAKAO_REST_API_KEY=your_kakao_rest_api_key
NEXT_PUBLIC_KAKAO_REDIRECT_URI=https://dplog.com/auth/kakao/callback
```

**서버 환경변수 (백엔드)**
```env
KAKAO_REST_API_KEY=your_kakao_rest_api_key
KAKAO_ADMIN_KEY=your_kakao_admin_key
KAKAO_REDIRECT_URI=https://dplog.com/auth/kakao/callback
```

## 🖥️ 프론트엔드 구현

### 1. 카카오 SDK 설치

```bash
npm install @types/kakao-js-sdk
```

### 2. Kakao SDK 초기화

**src/utils/kakaoSdk.ts**
```typescript
declare global {
  interface Window {
    Kakao: any;
  }
}

export const initKakaoSdk = () => {
  if (typeof window !== 'undefined' && window.Kakao) {
    if (!window.Kakao.isInitialized()) {
      window.Kakao.init(process.env.NEXT_PUBLIC_KAKAO_REST_API_KEY);
    }
  }
};

export const getKakaoAuthUrl = (): string => {
  const clientId = process.env.NEXT_PUBLIC_KAKAO_REST_API_KEY;
  const redirectUri = process.env.NEXT_PUBLIC_KAKAO_REDIRECT_URI;
  const state = generateRandomString(32); // CSRF 방지
  
  // state 값을 localStorage에 저장 (검증용)
  localStorage.setItem('kakao_oauth_state', state);
  
  return `https://kauth.kakao.com/oauth/authorize?response_type=code&client_id=${clientId}&redirect_uri=${encodeURIComponent(redirectUri)}&state=${state}`;
};

const generateRandomString = (length: number): string => {
  const chars = 'ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789';
  let result = '';
  for (let i = 0; i < length; i++) {
    result += chars.charAt(Math.floor(Math.random() * chars.length));
  }
  return result;
};
```

### 3. 카카오 로그인 컴포넌트

**src/components/auth/KakaoLoginButton.tsx**
```typescript
'use client';

import { useEffect } from 'react';
import { initKakaoSdk, getKakaoAuthUrl } from '@/utils/kakaoSdk';
import Image from 'next/image';

export default function KakaoLoginButton() {
  useEffect(() => {
    // 카카오 SDK 스크립트 로드
    const script = document.createElement('script');
    script.src = 'https://t1.kakaocdn.net/kakao_js_sdk/2.7.4/kakao.min.js';
    script.integrity = 'sha384-DKYJZ8NLiK8MN4/C5P2dtSmLQ4KwPaoqAfyA/DfmEc1VDxu4yyln9NgcHQ6X8Q0h';
    script.crossOrigin = 'anonymous';
    script.onload = () => {
      initKakaoSdk();
    };
    document.head.appendChild(script);

    return () => {
      document.head.removeChild(script);
    };
  }, []);

  const handleKakaoLogin = () => {
    const kakaoAuthUrl = getKakaoAuthUrl();
    window.location.href = kakaoAuthUrl;
  };

  return (
    <button
      onClick={handleKakaoLogin}
      className="w-full flex items-center justify-center px-4 py-3 border border-transparent text-sm font-medium rounded-md text-gray-700 bg-yellow-400 hover:bg-yellow-500 focus:outline-none focus:ring-2 focus:ring-offset-2 focus:ring-yellow-500 transition-colors"
    >
      <Image
        src="/kakao-icon.svg"
        alt="카카오"
        width={20}
        height={20}
        className="mr-2"
      />
      카카오로 로그인
    </button>
  );
}
```

## 🔧 백엔드 구현

### 1. 카카오 로그인 API

**필수 엔드포인트:**

1. **POST /v1/auth/kakao/callback**: 카카오 로그인 콜백 처리
2. **POST /v1/auth/kakao/link**: 기존 계정과 카카오 연동
3. **POST /v1/auth/kakao/unlink**: 카카오 연동 해제

### 2. 카카오 API 연동 흐름

1. **인증 코드로 토큰 요청**
   ```
   POST https://kauth.kakao.com/oauth/token
   Content-Type: application/x-www-form-urlencoded
   
   grant_type=authorization_code
   &client_id={REST_API_KEY}
   &redirect_uri={REDIRECT_URI}
   &code={AUTHORIZATION_CODE}
   ```

2. **토큰으로 사용자 정보 조회**
   ```
   GET https://kapi.kakao.com/v2/user/me
   Authorization: Bearer {ACCESS_TOKEN}
   ```

## 🔐 보안 고려사항

### 1. CSRF 방지
- `state` 파라미터로 요청 위조 방지
- 랜덤 문자열 생성 및 검증

### 2. 토큰 보안
- Access Token은 서버에서만 처리
- Admin Key는 서버 환경변수로 관리
