# 카카오 싱크(Kakao Sync) 구현 가이드

## 📋 개요

카카오 싱크는 사용자의 카카오계정 정보를 앱에 동기화하여 더 나은 사용자 경험을 제공하는 기능입니다. 사용자가 카카오계정에서 프로필 정보를 변경하면 앱에도 자동으로 반영됩니다.

---

## 🎯 카카오 싱크의 주요 기능

### 1. 자동 정보 동기화
- **프로필 정보**: 닉네임, 프로필 이미지 자동 동기화
- **이메일 정보**: 카카오계정 이메일 변경 시 자동 업데이트
- **계정 상태**: 계정 탈퇴, 일시정지 등 상태 변화 감지

### 2. 사용자 경험 향상
- 별도의 프로필 설정 불필요
- 일관된 사용자 정보 관리
- 카카오계정 변경사항 실시간 반영

## 🔧 카카오 디벨로퍼스 설정

### 1. 카카오 싱크 활성화

**카카오 디벨로퍼스 콘솔 > 제품 설정 > 카카오 싱크**

```
1. 카카오 싱크 > 활성화 설정 > ON
2. 동기화 정보 선택:
   - 프로필 정보 (닉네임, 프로필 이미지)
   - 이메일
   - 연령대
   - 성별
   - 생일
```

### 2. 동의항목 설정

**필수 설정:**
- 프로필 정보(닉네임/프로필 사진): 필수 동의
- 카카오계정(이메일): 필수 동의

**선택 설정:**
- 연령대: 선택 동의
- 성별: 선택 동의
- 생일: 선택 동의

### 3. Webhook URL 설정

**카카오 싱크 이벤트 수신용 Webhook URL 등록**
```
https://yourdomain.com/api/kakao/sync/webhook
```

### 2. 카카오 싱크 유틸리티

**src/utils/kakaoSync.ts**
```typescript
import { KakaoSyncUserInfo } from '@/src/types/kakaoSync';

// 카카오 싱크 사용자 정보 조회
export const getKakaoSyncUserInfo = async (accessToken: string): Promise<KakaoSyncUserInfo> => {
  const response = await fetch('https://kapi.kakao.com/v1/user/me', {
    method: 'GET',
    headers: {
      'Authorization': `Bearer ${accessToken}`,
      'Content-Type': 'application/x-www-form-urlencoded;charset=utf-8',
    },
  });

  if (!response.ok) {
    throw new Error('카카오 사용자 정보 조회 실패');
  }

  return response.json();
};

// 카카오 싱크 상태 확인
export const checkKakaoSyncStatus = async (userId: number, accessToken: string): Promise<boolean> => {
  try {
    const userInfo = await getKakaoSyncUserInfo(accessToken);
    return !!userInfo.synched_at;
  } catch (error) {
    console.error('카카오 싱크 상태 확인 실패:', error);
    return false;
  }
};

// 프로필 정보 비교 및 업데이트 필요 여부 확인
export const needsProfileUpdate = (localProfile: any, kakaoProfile: any): boolean => {
  const localNickname = localProfile?.nickname || '';
  const localProfileImage = localProfile?.profileImage || '';
  
  const kakaoNickname = kakaoProfile?.properties?.nickname || 
                       kakaoProfile?.kakao_account?.profile?.nickname || '';
  const kakaoProfileImage = kakaoProfile?.properties?.profile_image || 
                           kakaoProfile?.kakao_account?.profile?.profile_image_url || '';

  return localNickname !== kakaoNickname || localProfileImage !== kakaoProfileImage;
};
```

### 3. 카카오 싱크 컴포넌트

**src/components/auth/KakaoSyncManager.tsx**
```typescript
'use client';

import { useEffect, useState } from 'react';
import { useAuthStore } from '@/src/store/provider/StoreProvider';
import { getKakaoSyncUserInfo, needsProfileUpdate } from '@/src/utils/kakaoSync';

export default function KakaoSyncManager() {
  const { loginUser, setLoginUser } = useAuthStore();
  const [isSyncing, setIsSyncing] = useState(false);

  useEffect(() => {
    const syncKakaoProfile = async () => {
      if (!loginUser?.kakaoAccessToken) return;

      try {
        setIsSyncing(true);
        
        // 카카오에서 최신 사용자 정보 조회
        const kakaoUserInfo = await getKakaoSyncUserInfo(loginUser.kakaoAccessToken);
        
        // 로컬 정보와 비교하여 업데이트 필요 여부 확인
        if (needsProfileUpdate(loginUser, kakaoUserInfo)) {
          console.log('프로필 정보 업데이트 필요');
          
          // 백엔드에 프로필 업데이트 요청
          const response = await fetch('/api/user/sync-kakao-profile', {
            method: 'POST',
            headers: {
              'Content-Type': 'application/json',
              'Authorization': `Bearer ${loginUser.accessToken}`,
            },
            body: JSON.stringify({
              kakaoUserInfo: kakaoUserInfo,
            }),
          });

          if (response.ok) {
            const updatedUser = await response.json();
            setLoginUser(updatedUser);
            console.log('프로필 동기화 완료');
          }
        }
      } catch (error) {
        console.error('카카오 싱크 오류:', error);
      } finally {
        setIsSyncing(false);
      }
    };

    // 로그인 시 최초 1회 동기화
    syncKakaoProfile();

    // 주기적 동기화 (30분마다)
    const syncInterval = setInterval(syncKakaoProfile, 30 * 60 * 1000);

    return () => clearInterval(syncInterval);
  }, [loginUser, setLoginUser]);

  // UI는 렌더링하지 않음 (백그라운드 동기화)
  return null;
}
```

**카카오에서 전송하는 Webhook 요청 검증:**
```javascript
const crypto = require('crypto');

function verifyKakaoWebhook(signature, body, secret) {
  const expectedSignature = crypto
    .createHmac('sha256', secret)
    .update(body)
    .digest('hex');
  
  return signature === expectedSignature;
}
```

## ❓ FAQ

### Q1. 카카오 싱크를 사용하면 어떤 이점이 있나요?
A: 사용자가 카카오계정에서 프로필을 변경하면 앱에도 자동으로 반영되어 일관된 사용자 경험을 제공합니다.

### Q2. 모든 사용자에게 카카오 싱크가 적용되나요?
A: 카카오 로그인을 통해 가입한 사용자 중 카카오 싱크에 동의한 사용자에게만 적용됩니다.

### Q3. 카카오 싱크를 해제할 수 있나요?
A: 네, 언제든지 사용자 설정에서 카카오 싱크를 해제할 수 있습니다.

### Q4. 동기화는 얼마나 자주 이루어지나요?
A: 실시간 Webhook 이벤트와 주기적 동기화(30분)를 통해 최신 정보를 유지합니다.

### Q5. 개인정보는 안전하게 보호되나요?
A: 사용자 동의 하에만 필요한 최소 정보만 동기화하며, 모든 데이터는 암호화되어 안전하게 보호됩니다.
