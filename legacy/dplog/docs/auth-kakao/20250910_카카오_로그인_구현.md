# 카카오 로그인 구현 가이드

## 📋 개요

카카오 로그인을 기존 일반 로그인 시스템에 추가하여 사용자가 두 가지 방식으로 로그인할 수 있도록 구현합니다.

---

## 🎯 구현 목표

1. **기존 로그인 유지**: 현재 일반 로그인 시스템은 그대로 유지
2. **카카오 로그인 추가**: REST API를 통한 카카오 로그인 추가
3. **계정 연동**: 기존 사용자도 카카오 계정 연동 가능
4. **신규 사용자**: 카카오 로그인으로 신규 회원가입 가능

---

## 🔑 카카오 개발자 설정

### 1. 카카오 디벨로퍼스 앱 설정

**필수 설정 항목:**

1. **플랫폼 등록**
   ```
   Web 플랫폼: https://dplog.com
   ```

2. **카카오 로그인 활성화**
   - 상품 설정 > 카카오 로그인 > 활성화 설정 > ON

3. **Redirect URI 설정**
   ```
   https://dplog.com/auth/kakao/callback
   ```

4. **동의항목 설정**
   - 닉네임: 필수 동의
   - 프로필 사진: 선택 동의
   - 이메일: 필수 동의 (기존 계정 연동을 위해)

5. **앱 키 정보**
   ```
   REST API 키: 클라이언트에서 사용
   Admin 키: 백엔드 서버에서 사용 (보안 주의)
   ```

```typescript
declare global {
  interface Window {
    Kakao: any;
  }
}

export const initKakaoSdk = () => {
  if (typeof window !== 'undefined' && window.Kakao) {
    if (!window.Kakao.isInitialized()) {
      window.Kakao.init(process.env.NEXT_PUBLIC_KAKAO_REST_API_KEY);
    }
  }
};

export const getKakaoAuthUrl = (): string => {
  const clientId = process.env.NEXT_PUBLIC_KAKAO_REST_API_KEY;
  const redirectUri = process.env.NEXT_PUBLIC_KAKAO_REDIRECT_URI;
  const state = generateRandomString(32); // CSRF 방지
  
  // state 값을 localStorage에 저장 (검증용)
  localStorage.setItem('kakao_oauth_state', state);
  
  return `https://kauth.kakao.com/oauth/authorize?response_type=code&client_id=${clientId}&redirect_uri=${encodeURIComponent(redirectUri)}&state=${state}`;
};

const generateRandomString = (length: number): string => {
  const chars = 'ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789';
  let result = '';
  for (let i = 0; i < length; i++) {
    result += chars.charAt(Math.floor(Math.random() * chars.length));
  }
  return result;
};
```

### 4. 카카오 콜백 페이지

**src/app/auth/kakao/callback/page.tsx**
```typescript
'use client';

import { useEffect, useState } from 'react';
import { useRouter, useSearchParams } from 'next/navigation';
import { toast } from 'react-hot-toast';

export default function KakaoCallbackPage() {
  const router = useRouter();
  const searchParams = useSearchParams();
  const [isProcessing, setIsProcessing] = useState(true);

  useEffect(() => {
    const handleKakaoCallback = async () => {
      try {
        const code = searchParams.get('code');
        const state = searchParams.get('state');
        const error = searchParams.get('error');

        // 에러 처리
        if (error) {
          throw new Error(`카카오 로그인 취소: ${error}`);
        }

        // 필수 파라미터 검증
        if (!code || !state) {
          throw new Error('카카오 로그인 정보가 올바르지 않습니다.');
        }

        // CSRF 검증
        const savedState = localStorage.getItem('kakao_oauth_state');
        if (state !== savedState) {
          throw new Error('보안 검증에 실패했습니다.');
        }

        // 백엔드에 카카오 로그인 요청
        const response = await fetch('/api/auth/kakao/callback', {
          method: 'POST',
          headers: {
            'Content-Type': 'application/json',
          },
          credentials: 'include',
          body: JSON.stringify({
            code,
            state,
          }),
        });

        const result = await response.json();

        if (result.code === '0') {
          // 로그인 성공
          localStorage.removeItem('kakao_oauth_state');
          
          if (result.data.isNewUser) {
            toast.success('카카오 로그인으로 회원가입이 완료되었습니다!');
          } else {
            toast.success('카카오 로그인 성공!');
          }
          
          router.push('/dashboard'); // 또는 적절한 페이지로 리다이렉트
        } else {
          throw new Error(result.message || '카카오 로그인에 실패했습니다.');
        }
      } catch (error) {
        console.error('Kakao login error:', error);
        toast.error(error instanceof Error ? error.message : '카카오 로그인 중 오류가 발생했습니다.');
        router.push('/login'); // 로그인 페이지로 리다이렉트
      } finally {
        setIsProcessing(false);
      }
    };

    handleKakaoCallback();
  }, [router, searchParams]);

  if (isProcessing) {
    return (
      <div className="min-h-screen flex items-center justify-center">
        <div className="text-center">
          <div className="animate-spin rounded-full h-12 w-12 border-b-2 border-yellow-500 mx-auto"></div>
          <p className="mt-4 text-gray-600">카카오 로그인 처리중...</p>
        </div>
      </div>
    );
  }

  return null;
}
```

## 🔧 백엔드 구현

### 1. 카카오 로그인 서비스 클래스

**KakaoAuthService.java**
```java
@Service
public class KakaoAuthService {
    
    @Value("${kakao.rest-api-key}")
    private String kakaoRestApiKey;
    
    @Value("${kakao.admin-key}")
    private String kakaoAdminKey;
    
    @Value("${kakao.redirect-uri}")
    private String kakaoRedirectUri;
    
    private final RestTemplate restTemplate;
    private final UserService userService;
    
    public KakaoAuthService(RestTemplate restTemplate, UserService userService) {
        this.restTemplate = restTemplate;
        this.userService = userService;
    }
    
    // 카카오 토큰 요청
    public KakaoTokenResponse getKakaoToken(String code) {
        String url = "https://kauth.kakao.com/oauth/token";
        
        HttpHeaders headers = new HttpHeaders();
        headers.setContentType(MediaType.APPLICATION_FORM_URLENCODED);
        
        MultiValueMap<String, String> params = new LinkedMultiValueMap<>();
        params.add("grant_type", "authorization_code");
        params.add("client_id", kakaoRestApiKey);
        params.add("redirect_uri", kakaoRedirectUri);
        params.add("code", code);
        
        HttpEntity<MultiValueMap<String, String>> request = new HttpEntity<>(params, headers);
        
        return restTemplate.postForObject(url, request, KakaoTokenResponse.class);
    }
    
    // 카카오 사용자 정보 조회
    public KakaoUserInfo getKakaoUserInfo(String accessToken) {
        String url = "https://kapi.kakao.com/v2/user/me";
        
        HttpHeaders headers = new HttpHeaders();
        headers.setBearerAuth(accessToken);
        
        HttpEntity<String> entity = new HttpEntity<>(headers);
        
        return restTemplate.exchange(url, HttpMethod.GET, entity, KakaoUserInfo.class).getBody();
    }
    
    // 카카오 로그인 처리
    @Transactional
    public LoginResult processKakaoLogin(String code, String state) {
        try {
            // 1. 카카오 토큰 요청
            KakaoTokenResponse tokenResponse = getKakaoToken(code);
            
            // 2. 카카오 사용자 정보 조회
            KakaoUserInfo kakaoUser = getKakaoUserInfo(tokenResponse.getAccessToken());
            
            // 3. 기존 사용자 확인 (이메일 기준)
            User existingUser = userService.findByEmail(kakaoUser.getKakaoAccount().getEmail());
            
            if (existingUser != null) {
                // 기존 사용자 - 카카오 정보 업데이트
                existingUser.setKakaoId(kakaoUser.getId().toString());
                existingUser.setKakaoConnectedAt(LocalDateTime.now());
                userService.updateUser(existingUser);
                
                return LoginResult.builder()
                    .user(existingUser)
                    .isNewUser(false)
                    .build();
            } else {
                // 신규 사용자 - 카카오 정보로 회원가입
                User newUser = User.builder()
                    .email(kakaoUser.getKakaoAccount().getEmail())
                    .name(kakaoUser.getKakaoAccount().getProfile().getNickname())
                    .kakaoId(kakaoUser.getId().toString())
                    .kakaoConnectedAt(LocalDateTime.now())
                    .profileImageUrl(kakaoUser.getKakaoAccount().getProfile().getProfileImageUrl())
                    .build();
                
                User savedUser = userService.createUser(newUser);
                
                return LoginResult.builder()
                    .user(savedUser)
                    .isNewUser(true)
                    .build();
            }
        } catch (Exception e) {
            throw new RuntimeException("카카오 로그인 처리 중 오류가 발생했습니다.", e);
        }
    }
}
```

### 2. 카카오 로그인 컨트롤러

**KakaoAuthController.java**
```java
@RestController
@RequestMapping("/v1/auth/kakao")
public class KakaoAuthController {
    
    private final KakaoAuthService kakaoAuthService;
    private final SessionService sessionService;
    
    public KakaoAuthController(KakaoAuthService kakaoAuthService, SessionService sessionService) {
        this.kakaoAuthService = kakaoAuthService;
        this.sessionService = sessionService;
    }
    
    // 카카오 로그인 콜백 처리
    @PostMapping("/callback")
    public ResponseEntity<ApiResponse<Map<String, Object>>> kakaoCallback(
            @RequestBody KakaoCallbackRequest request,
            HttpServletRequest httpRequest,
            HttpServletResponse httpResponse) {
        
        try {
            // state 파라미터 검증 (CSRF 방지)
            if (request.getState() == null || request.getState().length() < 10) {
                return ResponseEntity.badRequest()
                    .body(ApiResponse.error("-2", "잘못된 요청입니다."));
            }
            
            // 카카오 로그인 처리
            LoginResult result = kakaoAuthService.processKakaoLogin(request.getCode(), request.getState());
            
            // 세션 생성
            String sessionId = sessionService.createSession(result.getUser(), httpRequest);
            
            // 세션 쿠키 설정
            Cookie sessionCookie = new Cookie("JSESSIONID", sessionId);
            sessionCookie.setHttpOnly(true);
            sessionCookie.setSecure(true);
            sessionCookie.setPath("/");
            sessionCookie.setMaxAge(86400); // 24시간
            httpResponse.addCookie(sessionCookie);
            
            // 응답 데이터 구성
            Map<String, Object> responseData = new HashMap<>();
            responseData.put("userId", result.getUser().getId());
            responseData.put("username", result.getUser().getEmail());
            responseData.put("name", result.getUser().getName());
            responseData.put("isNewUser", result.isNewUser());
            responseData.put("kakaoId", result.getUser().getKakaoId());
            responseData.put("sessionId", sessionId);
            
            return ResponseEntity.ok(ApiResponse.success(responseData, "카카오 로그인 성공"));
            
        } catch (Exception e) {
            return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR)
                .body(ApiResponse.error("-8", "카카오 로그인 처리 중 오류가 발생했습니다."));
        }
    }
    
    // 카카오 계정 연동 (기존 사용자용)
    @PostMapping("/link")
    public ResponseEntity<ApiResponse<Map<String, Object>>> linkKakaoAccount(
            @RequestBody KakaoCallbackRequest request,
            HttpServletRequest httpRequest) {
        
        try {
            // 현재 로그인된 사용자 확인
            User currentUser = sessionService.getCurrentUser(httpRequest);
            if (currentUser == null) {
                return ResponseEntity.status(HttpStatus.UNAUTHORIZED)
                    .body(ApiResponse.error("-3", "로그인이 필요합니다."));
            }
            
            // 카카오 토큰 요청 및 사용자 정보 조회
            KakaoTokenResponse tokenResponse = kakaoAuthService.getKakaoToken(request.getCode());
            KakaoUserInfo kakaoUser = kakaoAuthService.getKakaoUserInfo(tokenResponse.getAccessToken());
            
            // 현재 사용자에 카카오 정보 연동
            currentUser.setKakaoId(kakaoUser.getId().toString());
            currentUser.setKakaoConnectedAt(LocalDateTime.now());
            userService.updateUser(currentUser);
            
            Map<String, Object> responseData = new HashMap<>();
            responseData.put("linked", true);
            responseData.put("kakaoId", kakaoUser.getId().toString());
            
            return ResponseEntity.ok(ApiResponse.success(responseData, "카카오 계정 연동 완료"));
            
        } catch (Exception e) {
            return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR)
                .body(ApiResponse.error("-8", "카카오 계정 연동 중 오류가 발생했습니다."));
        }
    }
    
    // 카카오 계정 연동 해제
    @PostMapping("/unlink")
    public ResponseEntity<ApiResponse<Map<String, Object>>> unlinkKakaoAccount(
            HttpServletRequest request) {
        
        try {
            // 현재 로그인된 사용자 확인
            User currentUser = sessionService.getCurrentUser(request);
            if (currentUser == null) {
                return ResponseEntity.status(HttpStatus.UNAUTHORIZED)
                    .body(ApiResponse.error("-3", "로그인이 필요합니다."));
            }
            
            // 카카오 연동 해제
            currentUser.setKakaoId(null);
            currentUser.setKakaoConnectedAt(null);
            userService.updateUser(currentUser);
            
            Map<String, Object> responseData = new HashMap<>();
            responseData.put("unlinked", true);
            responseData.put("userId", currentUser.getId());
            
            return ResponseEntity.ok(ApiResponse.success(responseData, "카카오 계정 연동 해제 완료"));
            
        } catch (Exception e) {
            return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR)
                .body(ApiResponse.error("-8", "카카오 계정 연동 해제 중 오류가 발생했습니다."));
        }
    }
}
```

## 🔐 보안 고려사항
