## 🔧 useAuthGuard 훅 사용 패턴 분석

### 표준 패턴 (대부분의 페이지)
```typescript
export default function ComponentName() {
  const router = useRouter();
  const { isLoading, isGuest, isLogoutPending } = useAuthGuard();
  
  // 1. 로딩 상태 처리
  if (isLoading) {
    if (isLogoutPending) {
      return <LoadingFallback config={loadingUtils.logoutAuth()} />;
    }
    return <LoadingFallback config={loadingUtils.guestAuth()} />;
  }

  // 2. 게스트 사용자 처리
  useEffect(() => {
    if (isGuest && !isLogoutPending) {
      toast.error("로그인 후 이용해주세요.", {
        id: 'guest-redirect',
        duration: 3000
      });
      router.push("/login");
    }
  }, [isGuest, isLogoutPending, router]);

  if (isGuest) {
    return <LoadingFallback config={loadingUtils.loginRedirect()} />;
  }

  // 3. 메인 렌더링
  return <MainContent />;
}
```

### 발견된 개선 사항

#### 1. 코드 중복 최소화
**현재 상황**: 각 페이지에서 동일한 인증 로직이 반복됨

**개선 방안**: HOC(Higher-Order Component) 또는 커스텀 훅으로 추상화
```typescript
// 제안: withAuthGuard HOC
export function withAuthGuard<P extends object>(
  Component: React.ComponentType<P>
) {
  return function AuthenticatedComponent(props: P) {
    const { isLoading, isGuest, isLogoutPending } = useAuthGuard();
    const router = useRouter();

    // 인증 로직 처리
    if (isLoading) {
      if (isLogoutPending) {
        return <LoadingFallback config={loadingUtils.logoutAuth()} />;
      }
      return <LoadingFallback config={loadingUtils.guestAuth()} />;
    }

    useEffect(() => {
      if (isGuest && !isLogoutPending) {
        toast.error("로그인 후 이용해주세요.", {
          id: 'guest-redirect',
          duration: 3000
        });
        router.push("/login");
      }
    }, [isGuest, isLogoutPending, router]);

    if (isGuest) {
      return <LoadingFallback config={loadingUtils.loginRedirect()} />;
    }

    return <Component {...props} />;
  };
}
```

#### 2. 에러 처리 표준화
**개선 방안**: 전역 에러 처리 훅 생성
```typescript
// 제안: useErrorHandler 훅
export function useErrorHandler(error: Error | null, context: string) {
  useEffect(() => {
    if (error) {
      toast.error(`${context} 중 오류가 발생했습니다. 잠시 후 다시 시도해주세요.`, {
        id: `error-${context}`,
        duration: 5000
      });
    }
  }, [error, context]);
}
```
