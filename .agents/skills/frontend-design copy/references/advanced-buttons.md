# 고급 버튼 디테일 (Apple/SaaS 감성)

세련된 버튼은 아주 미세한 디테일에서 완성된다. 아래 체크리스트를 기본으로 적용한다.

## 핵심 기법

- **코너 라운딩**: 4~8px 미세 라운딩 또는 캡슐형(Pill). 가능하면 iOS 스타일의 부드러운 곡선을 연상시키는 연속 라운딩 느낌을 만든다.
- **다층 그림자**: 2~3겹의 낮은 불투명도(10~20%) + 넓은 블러로 깊이감. 버튼 메인 컬러를 아주 살짝 섞어 컬러 쉐도우를 만든다.
- **미세 그라데이션 + 내부 보더**: 거의 느껴지지 않는 상하 그라데이션을 적용하고, 배경보다 살짝 밝은 1px 내부 보더로 입체감을 만든다.
- **여백 비율**: 좌우 패딩을 상하 패딩의 약 2배로 설정해 안정감을 준다.
- **서체**: 버튼 크기에 비해 과하지 않은 Medium~Bold. 너무 두껍지 않게.
- **마이크로 인터랙션**: hover 시 색/광택 변화, active 시 0.98~0.99 스케일 다운.

## 요약 체크리스트

- 컬러: 원색보다 살짝 낮은 채도 + 깊이 있는 톤
- 그림자: 낮은 투명도 + 넓은 블러 + 컬러 틴트
- 보더: 1px 내부 하이라이트
- 모션: 클릭 시 미세 축소/완만한 색 변화

## 구현 팁

- **그라데이션**: 눈에 띄지 않을 정도로만 적용한다.
- **쉐도우**: 버튼 색상과 유사한 톤을 5~10% 섞으면 깊이감이 올라간다.
- **테스트**: 라이트/다크 배경에서 모두 확인한다.
