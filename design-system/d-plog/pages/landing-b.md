# Landing B (AI Personalization)

> ⚠️ **IMPORTANT:** Rules in this file **override** the Master file (`design-system/d-plog/MASTER.md`).

## 목표/포지션
- **목표:** 컨설팅/맞춤 리포트 전환
- **톤:** 전략적, 컨설팅형, 신뢰 중심
- **키 메시지:** "맞춤 전략을 설계합니다"

## 섹션 구조
1. Hero (개인화 메시지 + CTA)
2. Metrics (핵심 수치 4개)
3. Process (AI 진단 타임라인)
4. Dashboard Preview (개인화 카드)
5. Final CTA

## CTA 규칙
- 1차 CTA: **"맞춤 리포트 생성"** (Primary)
- 2차 CTA: **"샘플 리포트 보기"** (Secondary)
- 하단 CTA: **"1:1 컨설팅 신청"**

## 비주얼/레이아웃
- 히어로는 **센터 정렬** + 여백 넉넉
- Metrics 섹션은 4열 카드(모바일 2열)
- 타임라인은 좌측 라인 + 점으로 진행감 강조
- 배경: 밝은 화이트 톤, 섹션별 교차 배경

## 컴포넌트 규칙
- 카드: 텍스트 대비 강화, 여백 넓게
- 배지: "AI" / "LIVE" 같은 짧은 토큰만 사용
- 버튼: Primary는 CTA 컬러 고정

## 금지
- 히어로 내 과도한 수치 나열 금지
- 섹션 6개 이상 금지
