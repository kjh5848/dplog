# Landing A (Performance Funnel)

> ⚠️ **IMPORTANT:** Rules in this file **override** the Master file (`design-system/d-plog/MASTER.md`).

## 목표/포지션
- **목표:** 빠른 전환(무료 진단 시작)
- **톤:** 실무 중심, 데이터 기반, 즉시 실행
- **키 메시지:** "매장 순위를 올리는 가장 빠른 방법"

## 섹션 구조
1. Hero (문제 + 해결 + 1차 CTA)
2. Features (3개 카드)
3. 3-Step Funnel (문제 → 해결 → 행동)
4. Proof (수치 + 인용)
5. Pricing (3 tier)
6. Final CTA

## CTA 규칙
- 1차 CTA: **"무료 진단 시작"** (Primary)
- 2차 CTA: **"데모 보기"** (Secondary)
- CTA는 모든 섹션에서 동일한 문구 유지

## 비주얼/레이아웃
- 히어로 오른쪽에 **데이터 카드/스냅샷** 배치
- 카드 스타일: 라운드 12px, 얇은 보더, 낮은 그림자
- 배경: Master의 `--color-background` 유지
- 섹션 간 명확한 여백(48~64px)

## 컴포넌트 규칙
- 아이콘: Lucide (line icon), 크기 20~24px
- 카드: 제목 1줄 + 설명 2줄 이내
- 버튼: Primary는 CTA 컬러 고정

## 금지
- 히어로에 슬라이더/캐러셀 금지
- 카드 내 텍스트 3줄 초과 금지
